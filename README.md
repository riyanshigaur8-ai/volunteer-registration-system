# VolunteerHub

VolunteerHub is a full-stack volunteer management platform that connects volunteers with community service opportunities.

## Features

* User Registration & Login
* Secure Password Hashing
* Volunteer Dashboard
* Event Management
* Event Categories
* Join Events
* My Events Tracking
* Admin Dashboard
* Profile Management
* Protected Routes
* Role-Based Access Control (Admin & Volunteer)

## Tech Stack

### Frontend

* React.js
* Vite
* Tailwind CSS
* React Router

### Backend

* Flask
* SQLAlchemy
* Flask-CORS
* Bcrypt

### Database

* MySQL
